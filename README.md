## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation

1. Clone or download the repository
2. Open `index.html` in a web browser, or
3. Use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server -p 8000

# Using PHP
php -S localhost:8000
```

4. Navigate to `http://localhost:8000`

## üì± Usage

### Navigation
- **Swipe Up/Down**: Navigate between sections on mobile
- **Mouse Wheel**: Scroll between sections on desktop
- **Navigation Dots**: Click to jump to specific sections

### Features

#### Polls
1. Navigate to the Polls section
2. Click on your answer choice
3. View results after voting

#### Red Flags
1. Navigate to the Red Flags submission section
2. Type your red flag or select from quick picks
3. Submit to add to the community list
4. View all submitted red flags in the next section

#### Music Requests
1. Navigate to the Music section
2. Fill in song title and artist
3. Add optional dedication
4. Submit your request
5. Vote on other submissions

#### Photobooth
1. Navigate to the Photobooth section
2. Upload a photo or use your camera
3. Rotate/flip to adjust
4. Download or share your branded photo
